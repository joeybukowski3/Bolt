Tag Location Guide

ASUS equipment typically displays the model and serial data tag on a manufacturer rating plate. Look for a label that lists "Model" and "Serial" and check common locations by product type: inside doors, behind access panels, on the back panel, or near the compressor / condenser area.

How ASUS Encodes Manufacture Dates

ASUS serial numbers use a straightforward two-character date code at the start of the serial. The first character encodes the manufacture year using an alphabetic code, and the second character encodes the manufacture month.

The year code sequence runs A = 2010, B = 2011, C = 2012, and so on, skipping the letters I, O, and Q to avoid visual confusion with the digits 1, 0, and an ambiguous character. This means J = 2018, K = 2019, L = 2020, M = 2021, N = 2022, P = 2023, R = 2024, and S = 2025.

The month code uses digits 1â€“9 for January through September, and the letters A, B, C for October, November, and December respectively. This compact encoding gives ASUS a clear, unambiguous manufacture date without any decade-cycling ambiguity â€” each year code maps to exactly one calendar year.

Where to Find Your ASUS Serial Number

  • Laptops: Sticker on the bottom of the device
  • Desktops / All-in-ones: Label on the back or side panel
  • Motherboards: Printed on the motherboard PCB or on the anti-static packaging
  • Via Windows: Settings → System → About, or run wmic bios get serialnumber in Command Prompt
  • Via ASUS MyASUS app: The serial number is displayed on the Device Information page

Common Mistakes When Decoding ASUS Serials

Confusing skipped letters. The ASUS year sequence skips I, O, and Q. If your first character is I or O, you are likely reading the wrong field â€” I is not a valid ASUS year code. Check that you are reading the serial number, not the product number.

Using the part number instead of the serial number. ASUS products often have both a serial number (S/N) and a part number or model number on the same label. The serial number for date decoding is the one in the S/N field, not the model code (e.g., UX3402ZA).

Older products (pre-2010). Devices manufactured before 2010 may use a different or non-standardized serial format. If the first character does not appear in the year code table, use Smart Lookup for research.
